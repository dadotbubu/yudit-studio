📌 자주 쓰는 내용 — 목업 v3

메모 탭 상단. 탭 만들 때 "텍스트만 / 제목+내용" 형태 선택 → 항목 추가는 형태에 따라 자동.

📌 자주 쓰는 내용 (15)

↓ 답글 탭 (제목 없는 일반 항목)

↓ 링크 탭 (제목 4자 + 내용 항목, 4글자까지 풀 노출)

↑ "복사" 누르면 제목 ("공지", "자료실", "브랜딩") 빼고 https://… 부분만 클립보드로

현재 탭이 "텍스트만"이면 한 줄, "제목+내용"이면 두 칸으로 자동 추가

동작 정리

  • • 탭 클릭 → 해당 탭의 항목 표시
  • • 탭 옆 × → "이 탭 삭제하시겠습니까?" → 탭과 그 안의 항목 전부 삭제
  • • + 탭 → ① 이름 입력 → ② 형태 선택 (텍스트만 / 제목+내용)
  • • + 항목 추가 → 현재 탭의 형태에 맞춰 자동 (한 줄 or 두 칸)
  • • 제목 칸: 최대 4자, 한글 4자까지 잘림 없이 풀 노출 (≈ 76px)
  • • 복사 → 클립보드 (제목+내용 탭은 내용만) + "복사됨" 토스트
  • • 휴지통 → 확인 후 항목 삭제
  • • 모든 입력 → 400ms debounce 자동저장 (Supabase 동기화)
  • • 접기/펼치기 → 헤더 클릭 (기본 펼쳐짐)
